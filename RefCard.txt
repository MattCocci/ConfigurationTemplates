TMUX
=====

My Leader (^): Ctrl + a

Sessions
---------
tmux            New Session     
tmux new -s     Start new sesh with  
    myname          name "myname"
tmux a -t       Reattach named session
    myname     
tmux switch -t  Switch to sesh "myname"
    myname

tmux attach     Start saved session
tmux ls         Display sessions
^d              Detach session
^s              List sessions
^$              Rename session

Windows (i.e. Tabs)
--------------------
^w              List windows
^c              Create new window
^,              Rename current window

^{num}          Switch btw windows
^l              Go to last active window
^n              Go to next window
^p              Go to previous window
^&              Kill window

Panes
------
^"              Split horizontal
^%              Split vertical
^o              Switch to other pane
^k              Kill current pane
^!              Kill all but current pane
^q              Show pane number

^+              Grow horiz
^-              Shrink horiz
^*              Grow vert
^/              Shrink vert
^{              Move current pane left


Misc
-----
^t              Show time



VIM
====

Text Manipulation
------------------
I       Insert before first non-blank on line
gI      Insert column 1
D       Delete chars from cursor to $
R       Enter replace mode
di'     Delete within ('); works for {}, (), ""


Folding
--------------------------
<L>za   Open/close fold      
<L>zA   Open close ALL folds


Buffers
--------
:ls     List buffers



File Interaction, Vim Use
--------------------------
:e          Edit cur file; reload if chgd outside vim
gf          Open to file under cursor
:r file     Insert contents of "file" here
:reg        Show registers


Navigation
--------------
H,M,L       Jump to High, Middle, Low of page
g0          To 1st non-blank on vis line when wrapping
g$          To last non-blank on vis line when wrapping
f{char}     Go to 1st instance of {char} after cursor
t{char}     Go 1 before 1st instance of {char} after cursor
m{char}     Set mark at cursor with id {char}
`{char}     Go to mark with id {char}


Nerdcommenter
--------------
<L>cc       Comment out
<L>cu       Uncomment
<L>ci       Toggle individual line comments
<L>cA       Add comment single and enter insert mode

Misc
-----
<L><L>w     Call easy-motion
:Tab /X     Align on character X
